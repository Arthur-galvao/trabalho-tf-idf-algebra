# trabalho-tf-idf-algebra

Este projeto, desenvolvido para a disciplina de Álgebra Linear, demonstra a aplicação prática da técnica TF-IDF (*Term Frequency-Inverse Document Frequency*) e Similaridade de Cosseno para criar um sistema de recomendação de GPUs.

O script utiliza um dataset de placas de vídeo (GPUs) e aplica um processo detalhado de tratamento de dados para permitir que o usuário encontre recomendações relevantes usando linguagem natural.
